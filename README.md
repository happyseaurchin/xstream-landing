# Xstream Landing Page

Holding page for xstream.machus.ai - Narrative coordination system for shared imagination.

## Features

- Summary of what Xstream does
- Video embed (placeholder)
- QR code/link for Sqale sharing
- White paper viewer
- Short story viewer
- Links to interactive React modules

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

## Deployment

Connected to Vercel for automatic deployments on push to main.

## DNS Configuration

Point `xstream.machus.ai` CNAME to `cname.vercel-dns.com`
